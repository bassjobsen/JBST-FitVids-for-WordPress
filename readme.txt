== Description ==

This plugin makes videos responsive using the FitVids jQuery plugin on WordPress.
The plugin use the latest version of the [FitVids jQuery plugin](https://github.com/davatron5000/FitVids.js) and is suitable to run together with the [WP Defer loading](http://wordpress.org/plugins/wp-defer-loading/) Plugin.

Update for the original plugin [FitVids for WordPress](http://wordpress.org/plugins/fitvids-for-wordpress/) which includes:

- remove jQuery CDN install (its bundled with WordPress by default)
- latest version of FitVids jQuery plugin
- [properly load Javascript and set dependency](http://bassjobsen.weblogs.fm/wordpress-development-properly-load-javascript-set-dependency/) 



Contribute!
-----------
If you have suggestions for a new feature or improvement, feel free to contact us on [Twitter](http://twitter.com/JamedoWebsites). Alternatively, you can fork the plugin from [Github](https://github.com/bassjobsen/JBST-FitVids-for-WordPress/).

== Installation ==

1. You can download and install JBST FitVids for WordPress using the built in WordPress plugin installer. If you download JBST FitVids for WordPress manually, make sure it is uploaded to "/wp-content/plugins/JBST-FitVids-for-WordPress/".

2. Activate WP Defer Loading in the "Plugins" admin panel using the "Activate" link.
